Tập tài liệu để tham khảo khi viết phần **thực trạng Firm Performance** (hiệu quả
hoạt động doanh nghiệp) trong luận án. Người dùng cung cấp (2026-06).

| File | Loại | Dùng cho |

| `OECD-Digital.pdf` | Báo cáo OECD | Chuyển đổi số ↔ năng suất doanh nghiệp (liên hệ DAI/TCI) |
| `OECD-MAGIC-Industrial-Subsidies-Database.pdf` | CSDL OECD | Trợ cấp công nghiệp — bối cảnh thể chế/chính sách (ICRV regime) |
| `OECD-Working-Group-Bribery-2025-Annual-Report-{a,b,c}.pdf` | Báo cáo OECD | Tham nhũng/liêm chính — chất lượng thể chế (Institutional Theory) |
| `OECD-a5cfb5dd-en.pdf` | Báo cáo OECD (iLibrary) | *Cần mở để xác định tiêu đề chính xác* |

> Uppsala). Hai file mã OECD và `SOA-Company-Profile` cần xác minh mức độ liên quan
