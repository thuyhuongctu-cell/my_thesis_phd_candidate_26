# PROSPERO 2024 — Plantilla de campos

> Plantilla autoritativa para `prospero_form.py`.
> El formulario web vive en: https://www.crd.york.ac.uk/PROSPERO/

## Campos obligatorios

| # | Campo | Notas |
|---|-------|-------|
| 1 | Title | Debe incluir "systematic review" o "meta-analysis". |
| 2 | Original language title | Si el manuscrito final será en otro idioma. |
| 3 | Anticipated/actual start date | Fecha de inicio del cribado, no del registro. |
| 4 | Anticipated completion date | Estimación. |
| 5 | Stage of review at submission | Las búsquedas preliminares pueden estar hechas, pero **no** la selección final. |
| 6 | Named contact | Una persona. |
| 7 | Named contact address | Postal address. |
| 8 | Named contact phone | Opcional pero recomendado. |
| 9 | Organisational affiliation | Hospital/Universidad. |
| 10 | Review team members | Todos los autores. |
| 11 | Funding sources | "None" si no aplica. |
| 12 | Conflicts of interest | "None declared" si no aplica. |
| 13 | Collaborators | Equipo extendido. |
| 14 | Review question | PICO completo. |
| 15 | Searches | Bases + fechas + estrategia (resumen). |
| 16 | URL to search strategy | Si está publicada (GitHub, OSF). |
| 17 | Condition or domain | Enfermedad/tema. |
| 18 | Participants/population | Inclusión + exclusión claras. |
| 19 | Intervention(s) | I del PICO. |
| 20 | Comparator(s) | C del PICO. |
| 21 | Types of study | RCT, observacionales, etc. |
| 22 | Context | Setting, países. |
| 23 | Main outcome(s) | Outcomes primarios. |
| 24 | Additional outcome(s) | Secundarios. |
| 25 | Data extraction | Doble extracción, tool usado, resolución de discrepancias. |
| 26 | RoB assessment | Tool concreto (RoB 2, ROBINS-I, NOS, etc.). |
| 27 | Data synthesis | Narrative o MA; si MA, modelo (random/fixed). |
| 28 | Analysis of subgroups | Si aplica. |
| 29 | Type of review | "Systematic review" / "Meta-analysis" / "Scoping review" (este último va a OSF, no PROSPERO). |
| 30 | Language | Idiomas de búsqueda. |
| 31 | Country | País del equipo. |
| 32 | Other registration details | Si hay duplicidad. |
| 33 | Reference/URL for published protocol | Si lo hay. |
| 34 | Dissemination plans | Publicación, congresos. |
| 35 | Keywords | 5–10. |
| 36 | Existing review by same authors | Para evitar duplicidad. |
| 37 | Current review status | "Ongoing" en inicio. |

## Notas operativas

- **PROSPERO no acepta scoping reviews** — esas van en OSF (Open Science Framework).
- **Cambios al protocolo** tras registro se documentan como "Amendments". Se aceptan, pero hay que justificarlos.
- **Tiempo de aprobación** suele ser 2–4 semanas.
- **No registrar** si la RS ya tiene los resultados finales — se rechaza.
- **PRISMA 2020 + PRISMA-P (Preferred Reporting Items for Systematic review and Meta-Analysis Protocols)** son los estándares de protocolo.
