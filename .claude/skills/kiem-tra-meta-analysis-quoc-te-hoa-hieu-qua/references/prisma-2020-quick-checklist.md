# PRISMA 2020 Quick Reference Checklist\n\nSử dụng checklist này để kiểm tra nhanh tuân thủ PRISMA 2020. Đánh dấu [x] cho items đã đáp ứng.\n\n## Title and Abstract\n- [ ] 1. Title identifies study as systematic review/meta-analysis\n- [ ] 2. Structured abstract: background, objectives, methods, results, conclusions, registration\n\n## Introduction\n- [ ] 3. Rationale for review in context of existing knowledge\n- [ ] 4. Explicit statement of objectives/research questions (PICO)\n\n## Methods\n- [ ] 5. Protocol registration (PROSPERO, OSF) with registration number\n- [ ] 6. Eligibility criteria with justification\n- [ ] 7. Information sources (databases, dates, last search date)\n- [ ] 8. Full search strategy for ≥1 database\n- [ ] 9. Selection process (# reviewers, blinding, software)\n- [ ] 10. Data collection process (extraction form, # extractors, pilot)\n- [ ] 11. Data items (all variables extracted listed)\n- [ ] 11a. Outcome prioritization specified\n- [ ] 11b. Deviations from protocol noted\n- [ ] 12. Risk of bias assessment (tool, # assessors)\n- [ ] 12a. Outcome-level risk of bias if applicable\n- [ ] 13. Effect measures (r, d, OR, RR with justification)\n- [ ] 13a. Synthesis methods (fixed/random, software, missing data handling)\n- [ ] 13b. Reasons for not meta-analyzing if applicable\n- [ ] 13c. Moderator/subgroup analysis methods\n- [ ] 13d. Sensitivity analysis methods\n- [ ] 13e. Certainty assessment methods (GRADE)\n- [ ] 13f. Publication bias assessment methods\n\n## Results\n- [ ] 14. Study selection results (PRISMA flow diagram)\n- [ ] 15. Study characteristics table\n- [ ] 16. Risk of bias results\n- [ ] 16a. Outcome-level risk of bias results\n- [ ] 17. Results of syntheses (forest plot, effect size, CI, p, I²)\n- [ ] 17a. Heterogeneity statistics (Q, I², τ²)\n- [ ] 17b. Moderator analysis results\n- [ ] 17c. Sensitivity analysis results\n- [ ] 18. Publication bias assessment results (funnel plot + tests)\n- [ ] 19. Certainty of evidence (GRADE ratings)\n\n## Discussion\n- [ ] 20. Discussion of main findings\n- [ ] 20a. Strength of evidence for each outcome\n- [ ] 20b. Limitations at study and review level\n- [ ] 20c. Implications for practice/policy\n- [ ] 20d. Implications for future research\n\n## Other\n- [ ] 21. Funding sources and role\n- [ ] 22. Conflicts of interest\n- [ ] 23. Data/materials availability\n- [ ] 24. Code availability (if applicable)\n\n---\n\n## Common Missing Items in I-P Meta-Analyses\n\nBased on quality assessments, these items are frequently omitted:\n\n### High Priority (Often Missing)\n1. **Protocol registration** (Item 5): Chỉ ~30% I-P meta-analyses pre-register\n2. **Full search strategy** (Item 8): Thường chỉ list databases, không có search strings\n3. **Risk of bias assessment** (Items 12, 16): ~40% skip hoặc không systematic\n4. **Certainty assessment** (Items 13e, 19): GRADE ratings rarely reported\n5. **Code/data availability** (Items 23, 24): Transparency issue\n\n### Medium Priority (Sometimes Missing)\n6. **Inter-rater reliability** (Item 9): Cohen's kappa often not reported\n7. **Deviation from protocol** (Item 11b): Nếu có protocol\n8. **Sensitivity analysis** (Items 13d, 17c): ~50% không thực hiện\n9. **Publication bias tests** (Item 18): Funnel plot có, tests thống kê thiếu\n\n### Low Priority (Usually Present)\n10. PRISMA flow diagram (Item 14)\n11. Study characteristics table (Item 15)\n12. Forest plot (Item 17)\n\n---\n\n## PRISMA Extensions Relevant to I-P Research\n\n### PRISMA-S (Search Extension)\n- More detailed search reporting\n- Relevant nếu complex search strategy\n\n### PRISMA-P (Protocol Extension)\n- For publishing protocols\n- Recommended before conducting meta-analysis\n\n### PRISMA-Abstract\n- 12-item checklist for abstracts\n- Condensed version of main checklist\n\n---\n\n## Quick Diagnostic Questions\n\n**If answer is NO to any, flag as concern:**\n\n1. Is there a PRISMA flow diagram showing study flow? → Item 14\n2. Are ≥3 databases searched? → Item 7\n3. Is search date within last 2 years of publication? → Item 7\n4. Are ≥2 independent reviewers used? → Item 9\n5. Is heterogeneity reported (I², Q)? → Item 17a\n6. Is publication bias assessed? → Item 18\n7. Is a random-effects model used (if I² > 25%)? → Item 13a\n8. Are moderators theoretically justified? → Item 13c\n9. Is risk of bias assessed for included studies? → Items 12, 16\n10. Are limitations discussed? → Item 20b\n\n**Scoring:**\n- 10/10 YES: Excellent compliance\n- 8-9/10 YES: Good compliance\n- 6-7/10 YES: Acceptable with gaps\n- <6/10 YES: Major compliance issues
