# Bilingual Glossary — WBES / International Business Research

Project-specific terminology for translating P3 (Vietnam), P4 (Singapore),
and P5 (China) manuscripts to Vietnamese.  Use these translations
verbatim; deviations require justification in a translator's note.

## Statistical notation (kept as-is)

Greek letters (β, α, σ, χ, μ, ρ), function names (ln, log, exp), test
statistics (t, F, p, z, χ², R², N, n), inequalities (≤, ≥, ≠, ≈), and
math expressions inside $...$ blocks are NOT translated.  Keep these
verbatim from the English source.

## Direction-of-finding language

Vietnamese academic writing favours conservative, hedged claims, matching the
"associations rather than effects" convention adopted in the English source:

| English (hedged) | Vietnamese (hedged) |
|---|---|
| "is associated with" | "có liên hệ với" / "có mối liên hệ với" |
| "is positively associated with" | "có liên hệ thuận chiều với" |
| "is negatively associated with" | "có liên hệ ngược chiều với" |
| "supports the hypothesis that..." | "ủng hộ giả thuyết rằng..." |
| "the data are consistent with..." | "dữ liệu nhất quán với..." |
| "we observe..." | "chúng tôi quan sát..." |
| "no statistically distinguishable..." | "không có sự khác biệt có ý nghĩa thống kê..." |

Avoid stronger causal verbs ("gây ra", "tác động lên", "ảnh hưởng tới")
unless the English source explicitly uses causal language and the design
supports causal identification.
