# Terminology Guidelines for Academic Translation

This reference provides detailed guidance on handling specialized terminology in English-to-Vietnamese academic translation.

## Terminology Decision Tree

When encountering a technical term in the English source:

```
1. Is the term in the provided glossary?
   YES → Use the glossary translation exactly
   NO → Continue to step 2

2. Is the term a universally recognized acronym (e.g., GDP, R&D, CEO)?
   YES → Keep English acronym, provide Vietnamese explanation on first mention
   NO → Continue to step 3

3. Does the term have an established Vietnamese equivalent in academic literature?
   YES → Use the established Vietnamese term, note English in parentheses on first mention
   NO → Continue to step 4

4. Is the term a compound of simpler terms you can translate?
   YES → Translate component by component, verify result makes sense
   NO → Continue to step 5

5. Create a new translation:
   - Translate literally if meaning is transparent
   - Use descriptive phrase if literal translation is unclear
   - Consult domain experts if available
   - Flag for glossary addition
```

## Domain-Specific Patterns

### Economics and Management

**Productivity terms:**
- productivity dispersion → phân tán hiệu quả
- total factor productivity (TFP) → năng suất nhân tố tổng hợp (TFP)
- labor productivity → năng suất lao động
- allocative efficiency → hiệu quả phân bổ

**Firm characteristics:**
- firm heterogeneity → tính không đồng nhất doanh nghiệp
- firm-level data → dữ liệu cấp doanh nghiệp
- small and medium enterprises (SMEs) → doanh nghiệp vừa và nhỏ (SMEs)
- state-owned enterprises (SOEs) → doanh nghiệp nhà nước (SOEs)

**Digital transformation:**
- digital adoption → áp dụng công nghệ số
- digital intensity → cường độ số hóa
- digital maturity → mức độ trưởng thành số
- Industry 4.0 → Cách mạng công nghiệp 4.0

### Statistics and Econometrics

**Regression methods:**
- ordinary least squares (OLS) → bình phương tối thiểu thông thường (OLS)
- two-stage least squares (2SLS) → bình phương tối thiểu hai giai đoạn (2SLS)
- instrumental variable (IV) → biến công cụ (IV)
- fixed effects → hiệu ứng cố định
- random effects → hiệu ứng ngẫu nhiên

**Statistical measures:**
- standard error (SE) → sai số chuẩn (SE)
- confidence interval (CI) → khoảng tin cậy (CI)
- heteroskedasticity → phương sai thay đổi
- autocorrelation → tự tương quan
- multicollinearity → đa cộng tuyến

**Inference:**
- statistically significant → có ý nghĩa thống kê
- p-value → giá trị p
- null hypothesis → giả thuyết không
- alternative hypothesis → giả thuyết đối

### Research Design

**Data sources:**
- World Bank Enterprise Survey (WBES) → Khảo sát Doanh nghiệp Ngân hàng Thế giới (WBES)
- cross-sectional data → dữ liệu mặt cắt ngang
- panel data → dữ liệu bảng
- time series → chuỗi thời gian

**Sampling:**
- random sampling → lấy mẫu ngẫu nhiên
- stratified sampling → lấy mẫu phân tầng
- sample size → kích thước mẫu
- response rate → tỷ lệ phản hồi

**Validity:**
- internal validity → giá trị nội tại
- external validity → giá trị bên ngoài
- construct validity → giá trị cấu trúc
- robustness check → kiểm tra độ vững

## Handling Ambiguous Terms

Some English terms have multiple possible Vietnamese translations depending on context:

### "Performance"
- Financial context → "hiệu quả tài chính" or "kết quả hoạt động"
- Productivity context → "năng suất" or "hiệu suất"
- General context → "thành tích" or "kết quả"

Choose based on what is being measured. Maintain consistency within each document.

### "Impact"
- Causal research → "tác động" (preferred for regression analysis)
- Descriptive research → "ảnh hưởng"
- Policy context → "tác động" or "hiệu ứng"

Use "tác động" for quantitative causal claims, "ảnh hưởng" for qualitative or correlational claims.

### "Significant"
- Statistical context → "có ý nghĩa thống kê" (never shorten to just "có ý nghĩa")
- Substantive context → "đáng kể", "quan trọng", "ý nghĩa thực tiễn"

Always distinguish statistical from practical significance.

## Acronym Management

### When to Keep English Acronyms

**Keep English when:**
1. Universally recognized internationally (GDP, CEO, R&D, ICT, FDI)
2. Statistical/methodological standard (OLS, 2SLS, IV, PSM, DID)
3. Dataset/survey name (WBES, WDI, GEM)
4. Software/tool (SPSS, Stata, Python, R)
5. Journal/organization abbreviation (APJM, MIR, IJoEM)

**Format on first mention:**
"Vietnamese full term (English full term — ACRONYM)"

Example: "Khảo sát Doanh nghiệp Ngân hàng Thế giới (World Bank Enterprise Survey — WBES)"

### When to Create Vietnamese Acronyms

**Create Vietnamese when:**
1. Vietnamese term is widely used in local academic literature
2. Government policy or institutional standard mandates it
3. The concept is Vietnam-specific

Example:
- "doanh nghiệp vừa và nhỏ (DNVVN)" — but note that "SMEs" is also acceptable

### Mixed Approach

For some terms, both Vietnamese and English acronyms coexist:

"doanh nghiệp nhà nước (DNNN) hay state-owned enterprises (SOEs)"

Use the version most common in your target audience. For international submission, prefer English acronyms.

## Terminology Consistency Tools

### Maintaining Consistency Across Documents

When translating multiple related documents (e.g., dissertation chapters, paper series):

1. **Create a project-specific glossary** as you translate the first document
2. **Export key terms** to a spreadsheet: English | Vietnamese | Context | First Used In
3. **Search and verify** each new document against the glossary before finalizing
4. **Update glossary** when new terms emerge
5. **Share glossary** with co-authors, committee members, or reviewers

### Glossary Maintenance

A well-maintained glossary includes:

- **English term** (canonical form)
- **Vietnamese translation** (preferred)
- **Alternative translations** (if context-dependent)
- **Part of speech** (noun, verb, adjective)
- **Domain** (economics, statistics, management)
- **Example sentence** (English and Vietnamese)
- **First appearance** (document and section)
- **Notes** (usage guidelines, ambiguities)

Example entry:

```
English: productivity dispersion
Vietnamese: phân tán hiệu quả
Alternatives: độ phân tán năng suất (less common)
Part of speech: noun phrase
Domain: economics, productivity studies
Example EN: "Productivity dispersion across firms has increased over time."
Example VI: "Phân tán hiệu quả giữa các doanh nghiệp đã tăng theo thời gian."
First used: P3 Vietnam, Introduction section
Notes: Prefer "phân tán hiệu quả" for consistency with prior literature
```

## Special Cases

### Neologisms and Emerging Terms

For very recent English terms without established Vietnamese equivalents:

1. **Check recent Vietnamese academic publications** (2024-2026) in the same field
2. **Consult Vietnamese Wikipedia** for technical/scientific topics
3. **Use descriptive translation** if no consensus exists
4. **Note uncertainty** in a footnote or comment for reviewer awareness

Example:
- "algorithmic management" → "quản lý dựa trên thuật toán" (descriptive, transparent)

### Proprietary Names and Brands

**Keep original names** for:
- Software (Microsoft Excel, SPSS, Stata)
- Platforms (LinkedIn, Google Scholar)
- Organizations (World Bank, IMF, OECD)
- Datasets (Enterprise Surveys, World Development Indicators)

Provide Vietnamese explanation in parentheses on first mention if helpful:

"LinkedIn (mạng xã hội nghề nghiệp)"

### Methodological Schools and Frameworks

Some methodological terms reference specific scholars or schools:

- "Schumpeterian competition" → "cạnh tranh theo Schumpeter"
- "Granger causality" → "nhân quả Granger"
- "Herfindahl-Hirschman Index (HHI)" → "Chỉ số Herfindahl-Hirschman (HHI)"

Keep proper names in original form; translate only the generic component.

## Quality Assurance

Before finalizing terminology:

1. **Cross-check with published Vietnamese research** in the same domain
2. **Verify consistency** within the document (search for each English term)
3. **Confirm first-mention format** for all technical terms
4. **Test readability** with a Vietnamese academic colleague if possible
5. **Document any deviations** from standard translations with rationale

Terminology choices should prioritize:
1. **Accuracy** (correct meaning)
2. **Consistency** (same term throughout)
3. **Clarity** (understandable to target audience)
4. **Convention** (aligned with field standards)

When these priorities conflict, accuracy and consistency take precedence.
