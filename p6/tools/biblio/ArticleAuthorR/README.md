# Article-and-author-information-with-R-from-Scopus-OpenAlex-and-Web-of-Science
Here we provide a practical guide for downloading and analyzing academic publication and author data from Scopus, OpenAlex, and Web of Science (WoS) using R, including setup instructions, API access, and example codes.
More details can be found  in the link:
https://sites.google.com/view/fairness-in-science-in-eng/research-works


The research leading to these results was funded from the Hellenic 
Foundation for Research and Innovation (H.F.R.I.) 
under the Action "2nd Call for Research Projects H.F.R.I. 
for the support of faculty members and researchers" with 
the Project "Fairness in Science: How Increased Visibility 
and Rewards Shape Scientific Collaborations" (Project Number: 2988).
