  (do khối bảng kiểm định vững). Xem `CONDENSATION_PLAN_vi.md`.
- Định dạng cuối: Times New Roman 12, giãn dòng 1,3, khổ A4 (áp khi xuất bản docx cuối).
- Nguồn bản thảo đầy đủ: `../p3_vietnam_vi.md` (đang để nhãn JABS — cần đổi sang KTPT khi rút gọn).
